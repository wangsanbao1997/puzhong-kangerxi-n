# 葡眾康爾喜N 益生菌介紹網頁

單頁式響應式形象網站（One-Page RWD Landing Page），採 Modern Organic / Japandi 風格。
介紹葡眾「康爾喜（N）乳酸菌顆粒」與益生菌衛教資訊。

## 檔案結構
- `index.html` — 頁面內容（14 區塊，繁體中文）
- `style.css` — Japandi 風格與動畫
- `script.js` — 導覽、滾動淡入、數字跑動、FAQ 手風琴等互動
- `images/` — 主視覺與情境圖（`.jpg`，目前為佔位圖，請替換為實拍照）

## 本機預覽
```bash
python -m http.server 5500
```
開啟 http://localhost:5500

## 待補項目
1. 將 `images/` 內的佔位圖替換為同檔名的真實照片。
2. 將文中 `（預留填寫）` 與 CTA 的 `href="#"` 換成實際的 LINE／電話／表單連結。
